# Real-time embedded C on the BeagleBone (AM335x)

Bare-metal-adjacent C targeting the AM335x: a cross-compiling CMake
toolchain (ARM + x86_64), config-driven execution, memory-mapped
register access, and a dual-EPWMSS PWM controller as the flagship
project.
